# Bloque 01 — Historia de Java y JVM

## Conceptos
Java, JVM, Bytecode, Garbage Collector.

## Idea clave
Write Once, Run Anywhere. Java compila a bytecode que la JVM interpreta.

## Explicación
Java fue creado para ser portable. Compila a bytecode (.class), que es ejecutado por la JVM. Tiene un Garbage Collector para manejar la memoria.

## Comparación con C/C++/Python
A diferencia de C/C++, Java no compila a código máquina ni tiene punteros manuales. A diferencia de Python, no es puramente interpretado en tiempo de ejecución.

## Ejemplo en Java
```java
public class Main {
    public static void main(String[] args) {
        System.out.println("¡Hola, mundo!");
    }
}
```

## Errores comunes
- Usar punteros.
- Código fuera de clases.
